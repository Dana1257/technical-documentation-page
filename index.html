Strings

A string is a sequence of one or more characters that may include letters, numbers, or symbols.

A string is used to store, manipulate, and concatenate values.

Concatenate means to join 2 or more strings together using the + sign.

The extra space after the name Bart creates a space between the first name and the last name.

Example:
let firstName = "Bart " let lastName = "Simpson" console.log(firstName + lastName) // Bart Simpson
  • Strings can use single or double quotes
  • The start and end quotes need to be the same.
  • If there already quotes in the string or an apostrophe use opposite syntax (the other type of quotes)
Examples:
let firstName = "Bart' // Will not work let firstName = "Bart"// Works let example = 'Don't walk'// Will not work let example = "Don't walk" // Works let example = 'Don\'t walk'// Also works using the backslash (\) before the apostrophe as an escape character

If Else

An if else statement states that if a certain condition is true a certain action will happen or if it's false another action will happen.

  • Whatever is between the curly brackets will be translated to a boolean (true or false)
  • If the first statement is true it is executed, if it is false the second statement is excuted
Example 1: When the Variable is True

let covidVaccinated = true if (covidVaccinated === true){ console.log("You may come to the party!") } else { console.log("Please don't come to the party!") } // You may come to the party!



Example 2: When the Variable is False

let covidVaccinated = false if (covidVaccinated === true){ console.log("You may come to the party!") } else { console.log("Please don't come to the party!") } // Please don't come to the party!"

Switch Statements

A switch statement is used to replace long if/else statements (if/else statements with more than 4 or 5 values).

If the expression matches value 1 the code that matches value 1 will be executed. If the expression matches value 2 the code that matches value 2 will be executed.

A break after each piece of code is mandatory otherwise the next case(s) will be executed until there is a break. In the example below if there is no break after September, the next month October will be executed.

  • In the example below if it's September the console will show September and when it is October the console will automatically show October. There is no need to do anything else in this case.
Example:
const month = new Date().getMonth() switch(month){ case 0: console.log("It's January"); break; case 1: console.log("It's February"); break; case 2: console.log("It's March"); break; case 3: console.log("It's April"); break; case 4: console.log("It's May"); break; case 5: console.log("It's June"); break; case 6: console.log("It's July"); break; case 7: console.log("It's August"); break; case 8: console.log("It's September"); break; case 9: console.log("It's October"); break; case 10: console.log("It's November"); break; case 11: console.log("It's December"); break; default:console.log("Something went wrong!"); }

Functions

  • A JavaScript function is a block of code designed to perform a specific task (Source: w3schools.com).
  • A Javascript function is executed when is is invoked (gets called) (Source: w3schools.com).
Example:
function multiply(a,b){ console.log(a * b) } multiply(2,3) multiply(8,3) // 6 // 24

In the above example the function keyword is declared. Followed by the name of the function (multiply). In parentheses are the parameters; which may have names separated by commas. The code that needs to be executed by the function is placed inside curly brackets. In the example the function is executed when the name of the function and the arguments in the parentheses are invoked (gets called).

Functions also have a return keyword that ends the function and returns the value so we can store and use the result. In other words the return keyword completes the function and sends back the value that comes back after return to where it was called. With console.log it is not possible to do that because the result is not stored in the function because it is not sent back to where it was called.

Example:
function squared (x){ return x * x } console.log(squared(4)); let result = squared(4); console.log(result * result); // 16 // 256
  • In the example above we used the result (16) and multiplied it by itself.

Arrays

An array is a list of values. The values can be of a variey of data types, even within the same array. In other words multiple values are stored in one variable.

  • We start counting the items on the list from 0
  • In the example below "Full House" = [0]
Example 1: Making the List and Accessing Items From the List

const tvShows = ["Full House", "Party Of Five", "Law and Order", "Dawson's Creek"] // To access an item from a list: console.log(tvShows[1]) // Party Of Five



Example 2: To Find Out The Data Type
const tvShows = ["Full House","Party of Five",5] console.log(typeof tvShows[2]) console.log(typeof tvShows[0]) // number // string



Example 3: Adding Values to the End of the List Using the push method

const tvShows = ["Full House","Party of Five","Law & Order","Dawson's Creek"] console.log(tvShows.push("Who's the Boss","Married with Children")) console.log(tvShows) // (6)["Full House", "Party of Five", "Law and Order", "Dawson's Creek", "Who's The Boss", "Married With Children"]
